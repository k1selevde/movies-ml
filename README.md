### Установка

Обязательно: Python, pip, Nodejs, npm.

0) ```python3 -m venv env```
1) activate: ```env/scripts/activate``` (Windows)
2) cd djangoApi/
    ```pip install -r requirements.txt```
2) cd djangoApi/frontend: 
    ```npm install```
3) cd ..
    ```python manage.py runserver```
